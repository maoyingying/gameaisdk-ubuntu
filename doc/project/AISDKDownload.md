

## AISDK工具包下载

- 下载AISDK工具包代码

- 下载完成后，双击msi文件进行安装
- AISDK主要目录结构介绍：

```
|-----game_ai_sdk
|-----|-----bin---------------------------AISDK的主要执行脚本，可执行文件
|-----|-----build-------------------------存放编译脚本
|-----|-----cfg---------------------------AISDK的配置文件
|-----|-----data--------------------------AISDK的模型权重文件，以及加载的图片资源
|-----|-----doc---------------------------AISDK的使用文档
|-----|-----log---------------------------存放AISDK运行时日志
|-----|-----Modules-----------------------存放AISDK依赖的第三方模块
|-----|-----protocol----------------------数据传递的协议、规则
|-----|-----result------------------------存放AISDK运行游戏后的视频资料
|-----|-----src---------------------------存放AISDK源程序文件
|-----|-----|-----AgentAI-----------------AgentAI模块代码
|-----|-----|-----API---------------------API接口代码
|-----|-----|-----ImgProc-----------------UI和图像识别代码
|-----|-----|-----IOService---------------IO模块的代码
|-----|-----|-----ManageCenter------------MC模块的代码
|-----|-----|-----PlugIn------------------AI插件的代码
|-----|-----test--------------------------测试代码
|-----|-----tools-------------------------存放AISDK各类工具
|-----|-----README.md---------------------AISDK使用前编译参考文档
|-----|-----requirements_SDKTool.txt------记录SDKTool运行所需要的依赖包
|-----|-----requirements.txt--------------记录AISDK运行所需要的依赖包
```
